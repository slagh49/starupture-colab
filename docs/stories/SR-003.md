## User Story
En tant qu'utilisateur, je veux voir ma base sur une carte interactive 2D avec un terrain procédural afin de visualiser la disposition de mes bâtiments.

## Critères d'acceptation
- [ ] Canvas 2D plein écran avec terrain procédural (fbm, biomes alien)
- [ ] Terrain généré une seule fois sur offscreen canvas 512x512, réutilisé à chaque frame
- [ ] Overlay sombre pour lisibilité des entités
- [ ] Grille monde (lignes tous les 50 000 unités si zoom suffisant)
- [ ] Rectangle pointillé cyan pour la BaseZone (bounding box)
- [ ] Entités positionnées avec cercles colorés par catégorie (colors.ts)
- [ ] Labels nom + recette sur les machines
- [ ] Légende des catégories affichée sur la carte
- [ ] Responsive (resize canvas sur window resize)

## Notes techniques
- Utiliser `constants/mapConfig.ts` pour WORLD_BOUNDS et world2screen/screen2world
- Utiliser `constants/colors.ts` pour CAT_COLORS
- Ordre des couches : Terrain → Overlay → Grille → BaseZone → Entities → Labels
- Hook `useSaveData` pour fetch les données depuis l'API
- Composant `MapCanvas.tsx` orchestre les couches
