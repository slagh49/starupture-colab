## User Story
En tant qu'utilisateur, je veux zoomer, panner la carte et cliquer sur une entité pour voir ses détails afin de naviguer facilement dans ma base.

## Critères d'acceptation
- [ ] Zoom molette centré sur le curseur (min/max définis dans mapConfig)
- [ ] Pan par clic-drag (bouton gauche)
- [ ] Hover sur une entité : highlight glow + tooltip (nom, catégorie, recette)
- [ ] Clic sur une entité : sélection persistante avec panneau détail
- [ ] Panneau détail affiche : nom, gameId, catégorie, X/Y/Z, recette, infection, statut
- [ ] Clic dans le vide : désélection
- [ ] Curseur change selon le contexte (grab/pointer)

## Notes techniques
- Hook `useMapInteraction` pour gérer zoom, pan, hover, sélection
- Utiliser `screen2world` pour convertir les coordonnées souris
- Hit-test : distance euclidienne entre curseur monde et entité < seuil
- Shadow glow sur entité survolée/sélectionnée (EntityLayer)
- Composant Tooltip pour le hover
