## User Story
En tant qu'utilisateur, je veux des alertes visuelles sur les machines infectées ou inactives afin de repérer rapidement les problèmes dans ma base.

## Critères d'acceptation
- [ ] Ring rouge pulsant autour des entités avec infection > 0
- [ ] Intensité du ring proportionnelle au niveau d'infection
- [ ] Badge "OFF" gris sur les machines avec statut off
- [ ] Opacité réduite sur les entités off sur la carte
- [ ] Indicateur visuel dans le tableau de production (colonne statut colorée)
- [ ] Tooltip enrichi avec niveau d'infection si > 0

## Notes techniques
- EntityLayer.ts : ring infection avec animation pulse (alpha oscillant)
- Utiliser timestamp de useAnimation pour l'oscillation
- Couleur infection : #ff3030 (danger dans CAT_COLORS)
- Opacité entité off : 0.4
