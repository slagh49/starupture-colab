## User Story
En tant qu'utilisateur, je veux voir les flux de drones animés entre machines afin de comprendre les chaînes logistiques de ma base.

## Critères d'acceptation
- [ ] Flèches vertes animées (lineDashOffset) entre source et destination
- [ ] Flèche directionnelle au milieu du lien indiquant le sens du flux
- [ ] Label item transporté affiché au milieu du lien
- [ ] Compteur drones visible sur le lien
- [ ] Animation fluide via requestAnimationFrame
- [ ] Les liens se mettent à jour quand on change de session
