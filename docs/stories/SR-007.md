## User Story
En tant qu'utilisateur, je veux un tableau de production triable et filtrable afin de consulter l'état de toutes mes machines en un coup d'oeil.

## Critères d'acceptation
- [ ] Tableau listant toutes les entités de type machine/energy
- [ ] Colonnes : nom, catégorie, recette, statut (on/off), infection, position
- [ ] Tri par clic sur l'en-tête de colonne (asc/desc)
- [ ] Recherche textuelle (filtre sur nom et recette)
- [ ] Filtre par statut (on/off/all)
- [ ] Filtre par catégorie
- [ ] Clic sur une ligne sélectionne l'entité (partagé avec la carte)
- [ ] Badge coloré pour la catégorie
- [ ] Badge rouge pour infection > 0
- [ ] Badge gris pour statut OFF
