## User Story
En tant qu'utilisateur, je veux une minimap qui se centre sur l'entité sélectionnée dans le tableau afin de localiser rapidement une machine.

## Critères d'acceptation
- [ ] Canvas minimap à côté du tableau de production
- [ ] Se centre sur l'entité sélectionnée avec un rayon de 60 000 unités
- [ ] Affiche le terrain procédural à l'échelle de la minimap
- [ ] Affiche les entités visibles dans le rayon
- [ ] Affiche les liens drones en pointillés verts
- [ ] Croix de ciblage (+) sur l'entité sélectionnée
- [ ] Panneau détail sous la minimap (nom, type, X/Y/Z, recette, drones)
- [ ] Se met à jour automatiquement quand la sélection change

## Notes techniques
- MiniMap.tsx dans components/table/
- Réutiliser TerrainLayer pour le rendu terrain
- Rayon fixe de 60 000 unités monde
